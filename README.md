# MedsORM

Application for managing a medicaments database.  
Group work for the course "Ingeniería de la Información", Software Engineering, UMA.

## Technologies

- AngularJS
- Sequelize.js
- SQLite3

## Build & development

Run `node index.js` to launch the application.
